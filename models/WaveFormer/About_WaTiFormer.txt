
### About WaTiFormer ###


CausalConv1d : 좌우 패딩 생성 -> dilate conv 진행 -> 우측 결과 무시

WaveNetBlock : 두개 곱해짐 -> residual + skip

Inception_Block : 여러 커널수의 Conv2d -> stack 후 평균

TimesBlock : FFT(Batch,Channel 평균) -> top_k 별로 2D Inception -> 1D stack -> freq 가중치 softmax 곱하기 + residual

PositionalEncoding : Casual

WaTiBlock : Time -> Wave -> Transformer

WaTiFormer : input embedding -> PositionalEncoding -> WaTiBlocks(dilation 달라짐) -> Output(many to many)
